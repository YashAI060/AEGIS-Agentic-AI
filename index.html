    Design Notes (for your current stack)
    
        
    
          
    
            
    Hybrid Router
    
            
    Add a policy layer that chooses Gemini vs. Llama based on: privacy tag, latency budget, tool access, and confidence scoring. Log every routing decision for debugging.
    
          
    
          
    
            
    Memory Upgrade Path
    
            
    Move from JSON → SQLite → vector store. Keep JSON as an export/import format. Add schema + migrations and a “facts TTL” for volatile data.
    
          
    
          
    
            
    Security Hardening
    
            
    Face unlock should include liveness (blink/motion), a second factor fallback, and a safe lockout mode. Never store raw encodings unencrypted.
    
          
    
          
    
            
    Tool Safety
    
            
    Wrap every action in a permission gate (confirm / allowlist), dry-run previews, and structured “tool contracts” to avoid unintended OS automation.
